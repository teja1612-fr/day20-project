#  CI/CD Pipeline Project

## Project Overview

This project demonstrates a complete CI/CD pipeline using GitHub Actions, Docker, Docker Hub, and AWS EC2.

## What I Did

* Created a Python web application.
* Containerized the application using Docker.
* Pushed the Docker image to Docker Hub.
* Configured GitHub Actions for CI/CD automation.
* Deployed the application on an AWS EC2 instance.
* Automated deployment whenever code is pushed to GitHub.
* Exposed the application using port 8081.
* Verified the deployment through a live application URL.

## Tools Used

* Linux (WSL)
* Git & GitHub
* GitHub Actions
* Docker
* Docker Hub
* AWS EC2
* Flask (Python)

## Outcome

Successfully built and deployed a live application using an automated CI/CD pipeline.
